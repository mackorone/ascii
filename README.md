# ascii

Render webcam video to the terminal, built with [opencv-python](https://pypi.org/project/opencv-python/)

## Quick start

```
$ python3.8 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ python ascii.py
```
